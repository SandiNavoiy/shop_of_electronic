Платформа торговой сети электроники

Сеть должна представлять собой иерархическую структуру из 3 уровней:

Завод;
Розничная сеть;
Индивидуальный предприниматель.

Приложение retail_chain (торговая сеть):

Сеть (Link)
Контакты (Contacts)
Продукты (Products)

Стек технологий:
Python 3.8+
Django 3+
DRF 3.10+
PostgreSQL 10+
